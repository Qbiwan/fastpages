Data is first webscrapped from [Basketball Reference](https://www.basketball-reference.com/players/) >> Pandas is used to pivot data for player comparison in each stat category >> plot each stat using seaborn >> served via streamlit and deployed on heroku.  Git Repo is available [Here](https://github.com/Qbiwan/NbaPlayerCompare).

NBA Player Compare App is deployed live on Heroku [HERE](https://nba-player-compare.herokuapp.com/), and live on Streamlit-Sharing [HERE](https://share.streamlit.io/qbiwan/nbaplayercompare-streamlitshare).

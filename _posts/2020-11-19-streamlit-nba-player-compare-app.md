Data is first webscrapped from [Basketball Reference](https://www.basketball-reference.com/players/) >> Pandas is used to pivot data for player comparison in each stat category >> plot each stat using seaborn >> served via streamlit and deployed on heroku. NBA Player Compare App is deployed on Heroku live [HERE](https://nba-player-compare.herokuapp.com/)

A Markov Chain Puzzle

I came across a puzzle recently, and solved it using markov chain. Not sure if my solution is correct or there are other ways to solve it.

Jan 10, 2021 • Qbiwan • 8 min read

Question

Parsing the question

1)

Not much useful information except the game is played for 1 hr

2)

Assumption I made:
C1: Probability of correct = 0.8, Probability of incorrect = 0.2
C2: Probability of correct = 0.6, Probability of incorrect = 0.4
C3: Probability of correct = 0.4, Probability of incorrect = 0.6
C4: Probability of correct = 0.2, Probability of incorrect = 0.8

3)

4)

Expectation : (Prob_correct X Payoff_correct) + (Prob_incorrect X Payoff_incorrect)

Expection(C1): 0.8 X 100 + 0.2 X 0 = 80
Expection(C2): 0.6 X 100 + 0.4 X 0 = 60
Expection(C3): 0.4 X 100 + 0.6 X 0 = 40
Expection(C4): 0.2 X 100 + 0.8 X -50 = -20

5) 

2) If you do not answer the question correctly, you go to next category question.

Two ways on interpreting Rule 2

1st interpretation: if wrong drop to the next worse category.

2nd interpretion: You get to choose the next category if you go wrong.
However,this won't work because there will be no way of ending up at C4 if the contestant deploy the following strategy

If contestant is allowed to select any category if he answers wrongly, the game will not work.

Example:
At C1: if wrong chose next preferred category C2
At C2: if wrong chose next preferred category C3
At C3: if wrong chose next preferred category C2
Result: will never end up in C4 ever, totally avoided the penalty

So 1st interpretation is adopted, and we have the following markov chain

We first assume that C4 is a absorbing condition, and proceed to solve it using a system of equations. We will calculate the expected time to absorbtion.

Let $V(Ci)$ = Expected number of steps to reach $C4$

1) What is the expected number of steps that a contestant finally ends up in C4 if she starts from C4**?

$V(C4) = 0 $ since if one is at C4 the number of steps to reach C4 is zero.

2) What is the expected number of steps that a contestant finally ends up in C4 if she starts from C3**?

If a contestant starts from C3, she will take one step, then ends in in either C1 or C4.
The probability that she answers correctly from C3 and ends up in C1 is 0.4, and
the probability that she answers incorrectly from C3 and ends up in C4 is 0.6.

$V(C3) = 1 + 0.4*V(C1) + 0.6*V(C4) $

Substitue $V(C4) = 0 $ into the above and we get

$V(C3) = 1 + 0.4*V(C1)$

3) What is the expected number of steps that a contestant finally ends up in C4 if she starts from C2**?

If a contestant starts from C2, she will take one step, then ends in in either C1 or C3.
The probability that she answers correctly from C2 and ends up in C1 is 0.6, and
the probability that she answers incorrectly from C2 and ends up in C3 is 0.4.

$V(C2) = 1 + 0.6*V(C1) + 0.4*V(C3) $

4) What is the expected number of steps that a contestant finally ends up in C4 if she starts from C1**?

If a contestant starts from C1, she will take one step, then ends in in either C1 or C2.
The probability that she answers correctly from C1 and ends up again in C1 is 0.8, and
the probability that she answers incorrectly from C1 and ends up in C2 is 0.2.

$V(C1) = 1 + 0.8*V(C1) + 0.2*V(C2) $

We now have a system of equations to solve for $V(Ci)$ as we have 3 variables and 3 equations.

$V(C3) = 1 + 0.4*V(C1)$ ---(eq 1)
$V(C2) = 1 + 0.6*V(C1) + 0.4*V(C3) $ ---(eq 2)
$V(C1) = 1 + 0.8*V(C1) + 0.2*V(C2) $ ---(eq 3)

Workings

From the above, we have a system of equations to solve for $V(Ci)$

Sub (eq 1) into (eq 2)

$V(C2) = 1 + 0.6*V(C1) + 0.4*(1 + 0.4*V(C1)) $
$V(C2) = 1 + 0.6*V(C1) + 0.4 + 0.16*V(C1)) $
$V(C2) = 1.4 + 0.76*V(C1)$ ----(eq 4)

Sub (eq 4) into (eq 3)

$V(C1) = 1 + 0.8*V(C1) + 0.2*( 1.4 + 0.76*V(C1)) $
$V(C1) = 1 + 0.8*V(C1) + 0.28 + 0.152*V(C1)) $
$V(C1) = 1 + 0.8*V(C1) + 0.28 + 0.152*V(C1)) $
$V(C1) = 1.28 + 0.952*V(C1)$
$V(C1)(1-0.952) = 1.28$
$V(C1) = 1.28/(1-0.952)$
$V(C1) = 1.28/0.048$
$V(C1) = 26.666$ ---(eq 5)

Sub (eq 5) into (eq 1)

$V(C3) = 1 + 0.4*26.666$
$V(C3) = 1 + 10.66666$
$V(C3) = 11.66666$ ---(eq 6)

Sub (eq 5) into (eq 4)

$V(C2) = 1.4 + 0.76*26.666)$
$V(C2) = 1.4 + 20.2666$
$V(C2) = 21.666$

So we have $V(C1) = 26.666$ , $C(C2) = 21.666$ , $V(C3) = 11.666$.

This makes intuitive sense since C1 is furthest away from C4 and C3 is nearest to C4, so rightfully the nearer you are to C4 it should take you a lower number of steps.

Finally, since we start from C0, we have equal chance of going to C1,C2 and C3.

$ V(C0) = 0.333*V(C1) + 0.333*V(C2) + 0.333*V(C3) $
$ V(C0) = 0.333*26.666 + 0.333*21.666 + 0.333*11.666 $
$ V(C0) = 0.333(26.666+21.666+11.666)$
$ V(C0) = 0.333(59.999)$
$ V(C0) = 19.999$
$ V(C0) = 20$

End of Workings

Hence, the expected time to absorbtion is 20 steps. Since it takes 20 steps i.e 20 questions to reach C4, the first question to be asked from C4 is the next question, which is the 21nd question.

Ans: For any new player, Question 21 will be the first question asked from category C4

We see that no matter which question a contestant is at, the expected payoff is \$60.

From the linearity of expectation, we know that the expectation of a sum = the sum of individual expected values. Hence, we can add up the expectation at each question to get the total expectation for any contestant. Since a contestant is given 60 mins and she is able to answer one question per minute, her expectation is 60*60 = 3600

Ans:the expected payout for the participant at the end of her game assuming it takes 1 min to answer each question is \$3600
